## Чек-лист проверок:

### 1. Login
    - Успешный логин (валидные данные)
    - Пустой username
    - Пустой password
    - Ошибка при неверных данных

### 2. Cart
    - Добавление товара
    - Удаление товара
    - Проврека заголовка
    - Переход в checkout

### 3. Checkout Step One
    - Пустая форма
    - Только имя
    - Только фамилия
    - Успешное заполнение валидными данными 

### 4. Checkout Step Two
    - Проверка total
    - Кнопка "finish"

### 5. Checkout Complete
    - Отображение сообщения об успешном заказе
